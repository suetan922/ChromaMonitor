# ChromaMonitor

ChromaMonitor は、キャプチャした画面や画像ファイルをもとに、
配色の偏り・彩度/明度の分布・見え方を確認するためのチェックツールです。

## 主な機能
- 色相環
- 配色比率
- S-V散布図
- HSV / RGBヒストグラム
- グレースケール / 2値化 / 3値化
- スクイント / 反転表示
- エッジ検出 / フォーカスピーキング / サリエンシーマップ

## 動作環境
- Windows

## 使い方
1. 設定からキャプチャ対象のウィンドウを選択
2. Start を押して解析開始
3. 必要なウィンドウを表示して確認
