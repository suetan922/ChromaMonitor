# ChromaMonitor

リアルタイムの画面キャプチャ/画像入力を解析し、色相環・散布図・ヒストグラムなどをドック型UIで確認するツールです。

※このREADMEは暫定版です。機

## 主な機能

- 色相環表示
- S-V散布図表示（色相フィルター付き）
- H/S/Vヒストグラム表示
- R/G/Bヒストグラム表示
- エッジ検出、グレースケール、2値化、3値化
- サリエンシーマップ、フォーカスピーキング、スクイント、ベクトルスコープ
- ドックの移動、分割、タブ化、フローティング

## 動作環境

- Python 3.10+
- Windows 10/11 推奨
- Linux/macOS は未検証

## 起動

```bash
python main.py
```

または:

```bash
python -m chroma_monitor
```

## 使い方

1. アプリを起動する
2. 設定 > 設定ウィンドウを開く > キャプチャで領域選択を行う
3. `ウィンドウ` メニューから表示したいビューをON/OFFする
4. ドックを並べ替えて作業しやすいレイアウトにする
5. Start ボタンを押して開始する

## 設定ファイル

- 設定は `config/settings.json` に保存されます
- レイアウト情報も同ファイルに保存されます
